# recipients — 대량 발송 수신자 목록

`send_bulk.py`(각 호 폴더)가 **이 폴더의 CSV/XLSX/TXT 에서 이메일을 자동 추출**해
BCC 한 통으로 발송합니다.

## 사용법

1. 엑셀 목록을 준비합니다. **열 위치·제목은 상관없습니다** — 어느 칸이든
   `a@b.com` 형태면 스크립트가 알아서 다 찾아냅니다.
2. 엑셀에서 **`파일 ▸ 다른 이름으로 저장 ▸ CSV UTF-8(쉼표로 분리)`** 로 저장해
   이 폴더(`recipients/`)에 넣습니다.
   - 엑셀(.xlsx)을 그대로 넣어도 됩니다. (이 경우 `openpyxl` 필요 — `run.bat`이 자동 설치)
3. 발송할 호 폴더에서 실행:
   ```
   py send_bulk.py
   ```
   → 이메일 추출 결과(고유 인원수/중복 제거 수) 출력
   → 전원이 **BCC** 에 들어간 Outlook 새 메일 창이 열림
   → **BCC 인원수 확인 후** 직접 **보내기**

## 동작 규칙

- 이 폴더 안의 **모든** CSV/XLSX/TXT 를 읽어 **합치고 중복 제거**(대소문자 무시)합니다.
- 엑셀 임시 잠금파일(`~$...`)은 무시합니다.
- **이름이 `_` 로 시작하는 파일은 읽지 않습니다** — 샘플(`_recipients_sample.csv`)이나
  잠시 빼두고 싶은 목록은 앞에 `_` 를 붙이면 발송에서 제외됩니다.
- 인코딩(UTF-8/CP949)은 자동 판별합니다.

## ⚠️ 개인정보 주의
